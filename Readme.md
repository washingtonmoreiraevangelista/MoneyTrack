#  MoneyTrack

Sistema de controle de gastos pessoais onde o usuário pode cadastrar e gerenciar suas despesas de forma simples.

---

##  Funcionalidades

- Cadastro de usuário
- Login
- Adicionar gastos
- Editar e excluir gastos
- Visualizar total de despesas
- Dashboard com resumo financeiro

---

## 🛠️ Tecnologias

### Backend
- Node.js
- Express
- TypeScript


### Frontend
- React
- Tailwind css
